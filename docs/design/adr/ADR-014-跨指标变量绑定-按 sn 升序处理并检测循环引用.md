## ADR-014：跨指标变量绑定，按 sn 升序处理并检测循环引用

| 项目 | 内容 |
|---|---|
| **上下文** | 某些计算规则需要引用其他指标的原始值或得分（如"入司天数决定权重区间"），但指标间可能存在循环依赖。 |
| **决策** | 支持四种变量格式（`${val}`、`${attr.xxx}`、`${dim.xxx}`、`${idx.指标编码.value/score}`），指标按 `dr_model_index.sn` 升序处理，仅允许引用 sn 更小的已完成指标，并做循环引用检测。 |
| **备选方案** | ① 不支持跨指标引用，所有依赖通过维度属性间接获取；② 支持任意顺序引用，用拓扑排序解析依赖 |
| **影响** | sn 顺序即依赖顺序，简单直观；循环引用在运行时检测并报错，防止无限递归；代价是配置时必须注意 sn 编排 |
